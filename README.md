# Daily Spend Tracker

This is a free personal iPhone-friendly spending tracker. It stores data locally in your browser.

## Use on iPhone like an app

For the best free no-code setup, upload this folder to a free static host such as GitHub Pages, Netlify, Cloudflare Pages, or Vercel. Then:

1. Open the hosted web address in Safari on your iPhone.
2. Tap Share.
3. Tap Add to Home Screen.
4. Open Daily Spend from your Home Screen.

You can also open `index.html` directly on a computer for testing. iPhone offline install works best from a hosted `https://` web address.

## Notes

- No App Store account is required.
- No Apple Developer Program payment is required.
- Your data stays local unless you use Export.
- For full offline install behavior, open it from a web address rather than a raw local file.

## GitHub Pages

This folder is ready to publish with GitHub Pages. Use the `main` branch and the repository root as the Pages source.
